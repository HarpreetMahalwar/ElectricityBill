# ElectricityBill

A simple Python console application that calculates an electricity bill based on electricity consumption, senior citizen discounts, and late payment penalties.

## Features

- Calculates bill based on units consumed
- Different pricing slabs:
  - 0–100 units → ₹5/unit
  - 101–300 units → ₹7/unit
  - Above 300 units → ₹10/unit
- 10% discount for senior citizens
- 2% late payment penalty per day
- Input validation for negative values
- Displays a detailed bill summary

## Technologies Used

- Python 3
